   - 또는 **시크릿 모드**로 테스트

2. **URL 접속**
   ```

   ```

3. **개발자 도구 열기** (F12)
   - Console 탭 확인
   - 에러 메시지 확인

4. **페이지 하단으로 스크롤**
   - 약관 동의 섹션 찾기

   - **확인**: 아래 3개 약관이 모두 자동으로 체크됨
   - **Console 확인**: 
     ```

       ✓ 체크박스 업데이트: agreeTerms = true

       ✓ 체크박스 업데이트: agreeMarketing = true
     ```

6. **에러 메시지 확인**
   - `custom.js` 에러가 있다면: 브라우저 확장 프로그램 비활성화 후 재테스트

## 🚨 문제 해결 가이드

### custom.js 에러가 계속 발생하는 경우

1. **브라우저 확장 프로그램 비활성화**
   - Chrome 확장 프로그램 모두 비활성화
   - 시크릿 모드에서 테스트

2. **브라우저 캐시 완전 삭제**
   - `Ctrl+Shift+Del`
   - "전체 기간" 선택
   - "캐시된 이미지 및 파일" 체크
   - 삭제 후 브라우저 재시작

3. **다른 브라우저로 테스트**
   - Chrome → Firefox
   - 또는 Edge 사용

4. **네트워크 탭 확인** (F12 → Network)
   - `signup` 파일이 정상 로드되는지 확인
   - 상태 코드: 200 OK 확인
