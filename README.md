# 알비(ALBI) - 1시간 체험 알바

## 🚀 프로젝트 개요

AI가 추천하는 내 주변 알바 플랫폼. 1시간 체험 후 결정하는 혁신적인 구인구직 서비스입니다.

### ✨ 주요 기능

#### 1. **실용적인 메인 페이지**
- ✅ **대형 검색창**: 지역, 업종, 회사명으로 즉시 검색
- ✅ **빠른 태그**: #편의점, #카페, #배달, #단기, #주말, #급구
- ✅ **업종별 카테고리**: 6개 카테고리 + 공고 건수 표시
- ✅ **실제 공고 미리보기**: 3개 샘플 카드 (거리, 시급, 배지)

#### 2. **친구초대 시스템**
- ✅ 상단 네비게이션에 친구초대 버튼 활성화
- ✅ +20P 포인트 보너스 배지 표시
- ✅ 리페럴 코드 기반 초대 시스템 (개발 예정)

#### 3. **완전한 로그인 시스템**
- ✅ **이메일/휴대폰 로그인**
- ✅ **간편 로그인**: 카카오, 네이버, Google
- ✅ 로그인 상태 유지 옵션
- ✅ 아이디/비밀번호 찾기 링크
- ✅ 깔끔한 UI/UX

#### 4. **완전한 회원가입 시스템**
- ✅ **휴대폰 인증 회원가입**
  - 6자리 인증번호 발송
  - 3분 타이머 카운트다운
  - 재전송 기능
- ✅ **간편 회원가입 (3초)**
  - 카카오로 시작하기
  - 네이버로 시작하기
  - Google로 시작하기
- ✅ **필수 입력 사항**
  - 이름, 휴대폰번호, 비밀번호
  - 이용약관, 개인정보처리방침 동의
- ✅ **선택 입력 사항**
  - 이메일, 마케팅 정보 수신 동의
- ✅ 실시간 유효성 검사
- ✅ 비밀번호 확인 매칭

#### 5. **검색 기능 활성화**
- ✅ Enter 키로 검색 실행
- ✅ 검색 버튼 클릭으로 검색
- ✅ 빠른 태그 클릭으로 즉시 검색
- ✅ `/jobs?keyword=검색어` 형식으로 이동

#### 6. **통일된 알비 캐릭터**
- ✅ `/albi-mascot.svg` 파일로 전체 통일
- ✅ 메인 페이지, 로그인, 회원가입에 동일 이미지 사용
- ✅ 일관된 브랜드 아이덴티티

## 🎨 디자인 컨셉

### 참고 사이트
- **사람인**: 깔끔한 네비게이션 + 공고 카드 레이아웃
- **급구**: 대형 검색창 + 빠른 태그 검색
- **동네알바/알바천국**: 카테고리 그리드 + 시급 명확 표시

### 디자인 특징
- ✅ 실용적이고 정보 밀도 높은 레이아웃
- ✅ 흰색 베이스 + 오렌지 포인트 컬러
- ✅ 명확한 공고 정보 (회사명, 위치, 시급, 거리)
- ✅ 배지 시스템 (1시간 체험, 급구, 인기)

## 📱 페이지 구조

### 메인 페이지 (`/`)
- 상단 네비게이션 (로고, 메뉴, 친구초대, 로그인, 회원가입)
- 대형 검색 섹션 (그라데이션 배경)
- 업종별 카테고리 (6개)
- 추천 공고 (3개 샘플)
- 특징 섹션 (1시간 체험, AI 매칭, 알비포인트)
- 푸터 (법적 정보, 고객센터)
- 모바일 하단 네비 (홈, 알바찾기, 커뮤니티, MY)

### 로그인 페이지 (`/login.html`)
- 알비 로고 + 타이틀
- 이메일/휴대폰 + 비밀번호 입력
- 로그인 상태 유지 체크박스
- 로그인 버튼
- 간편 로그인 (카카오, 네이버, Google)
- 아이디/비밀번호 찾기 링크
- 회원가입 링크

### 회원가입 페이지 (`/signup.html`)
- 알비 로고 + 타이틀
- **간편 회원가입** (상단)
  - 카카오로 3초만에 시작
  - 네이버로 3초만에 시작
  - Google로 3초만에 시작
- 구분선
- **휴대폰 인증 회원가입** (하단)
  - 이름 입력
  - 휴대폰번호 + 인증요청 버튼
  - 인증번호 입력 (6자리) + 타이머
  - 이메일 (선택)
  - 비밀번호 + 확인
  - 약관 동의 (전체 동의, 필수 2개, 선택 1개)
  - 회원가입 버튼
- 로그인 링크
